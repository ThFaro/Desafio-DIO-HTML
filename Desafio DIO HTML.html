Primeiros Computadores e Saltos Tecnológicos

A historia dos computadores pode ser dividida em gerações e no momento podemos dizer que estamos vivenciando a 5 dessas gerações

A primeira geração foi marcada por maquina enormes que funcionavam por válvulas e diversos fios, além de serem lentos e esquentarem de mais.

Alguns dos principais computadores desta geração são:

Harvard Mark 1 (1944):

Colossus (1946):

ENIAC(1946):

A segunda geração trocou as válvulas pelos primeiros modelos de transistores e os fios por circuitos impressos.

Já a terceira geração trouxe um salto tecnológico através dos:

- Transistores;

- Circuito integrado;

- Resistores;

- Capacitores.

Essas evoluções tecnológicas tornaram os computadores mais eficientes, compactos e capazes de realizar outros tipos de tarefas além de cálculos.

A quarta geração (1971), foi marcada pela introdução dos PCs e pela criação dos microprocessadores.

A quinta geração é a que vivemos atualmente, marcada pela IA, redes neurais, realidade aumentada dentre outras tecnologias de ponta.

Internet - Como surgiu

Antes da internet, o primeiro meio de comunicação intercontinental foi feito através do telégrafo (1858), e para isso foi instalado um cabo

ubmarino com quilômetros de distância entre a Irlanda e o Canadá.

Em 1950 durante a guerra fria o Estados Unidos criou a DARPA(Defense Advanced Research Projects Agency) utlizando o ENIAC para calculos balísticos,

como ferramenta para tentar vencer a guerra com a ajuda da tecnologia.

Neste período Joseph Licklider plantou a semente da ideia sobre uma rede de comunicação "intergalática" e a aprtir desse momento iniciaram-se pesquisas

voltadas ao assunto em questão.

No ano de 1969 surge então a "mãe" da internet como conhcemos hoje: a ARPANET.

A ARPANET foi criada para uso exclusivo militar, e transmitia pacotes de um a um, e permitia o envio para multiplos destinos.

Através do concetio da ARPANET, diversos micro sitemas foram criados até que em 1989, Tim Berners-Lee criou a Wolrd Wide Web (WWW).

Conceito de Clients no "linguajar Web"

É o dispositivo (PC, celular, tablet, etc) que consome determinado conteúdo na internet, enviando requisições e recebendo informações de

outro computador (servidor).Quando acessamos nosso navegador, enviamos requisições para o servidor e ele nos retorna arquivos HTMLm CSS, imagens,

dentre outro tipos conhecidos como arquivos estáticos. Quando o servidor nos envia os arquivos, o navegador gera os caches, armazenando

esses arquivos no dispositivo do usuário, para quando o acesso a esta mesma página for feito, as informações já esteja pré-carregadas para tornar

a experiência mais ágil. Outra ponto importante quando falamos do Client-side, é quanto ao design responsivo, ou seja, a experiência do usuário deve ser

intuitiva, de fácil navegação e compatível com tipo de dispositivo.

(Voltar)

Falando sobre Navegadores

Também chamados de browser , são programas gratuitos que seguem padrões da W3C responsáveis por interpretar, abrir e executar arquivos advindos da internet.

Seu início foi através do MOSAIC, seguido pelo NETscape e depois por todos os outros navegadores que conhecemos hoje em dia.

(Voltar)

Aplicações Web

São soluções criada para que possamos rodar determinado aplicativos sem a necessidade da sua instalação, rodando através do browser. Um bom exemplo disso

é o Facebook: Você pode instala-lo na sua máquina, ou pode simplesmente abrir o navegador, entrar no site do Facebook e fazer seu login e senha,

caracterizando assim uma aplicação web.

Dispositivos Móveis

Os dispositivos móveis estão cada vez mais presentes na nossas vidas, seja através de celulares, tablets, ou outros aparelhos. Por isso é muito importante

os levar em consideração no momento do desenvolvimento de uma página web, através de designs responsivos.

Sobre Servers

São computadores conectados a internet, que pode ser acessados desde um acesso a internet, quanto através de uma rede local. Ele deve ser uma máquina mais robusta,

já que precisa ”servir” diversas máquinas.Os servidores podem ser de diversos tipos como: Arquivos, Firewall, streaming, e-mail, web, etc.

(Voltar)

Hardwares e Softwares em Servidores

Todo computador é divido em softwares e hardwares, e com os servidores não é diferente. Os softwares que compõe um servidor são: sistema operacional, monitoramento

e servidores web. Já a parte de hardware é composta por: armazenamento, memória e processador.

Existem diversos sistemas operacionais para servidores, onde os mais usados são: Windows Server (específico para servidores) e alguns baseados em Linux como o: Ubuntu,

CentOS, Debian, Fedora e Oracle Linux.

Para que a comunicação com um web site acontece, este precisa ter um aplicação de apoio (servidor web) ao sistema operacional. O sistema operacional vai executar

um programa específico que farra o processamento de determinada requisição. Os servidores webs mais utilizado são o Apache e o Nginx.

(Voltar)

Tipos de Servidores

Servidor Proxy: É um tipo de servidor que consegue criar restrições de acesso para cada usuário, através de regras criadas. Também pode ser um servidor de autenticação.

Firwall: É voltado para acessos externos, bloqueando conteúdos maliciosos.

Web Server: Podemos sitar o Apache o Nginx

E-mail Server: Voltado especificamente para e-mails

Database Server: Responsável por armazenar o banco de dados.

Servidor Web sem Internet (Intranet): É acessados somente dentro da sua rede local através de um site (sem acesso a internet).

Servidor DNS(Domain Name Service): Armazena e traduz o “”www…” através de uma lista de nomes, e os redirecionada ao ip correspondente.

(Voltar)

Hospedagem em Sites

Significa colocar os seus arquivos do site em um servidor disponível para outros usuário acessarem. Esse servidor pode ser contratado

ou criado por você mesmo, mas o mais comum é contratar esse tipo de serviço.

Os servidores mais completos são uso multiplataformas que englobam no mesmo plano o DNS, e-mail, certificado ssl, etc.

Para subir seu site para um servidor é necessário um conexão FTP (File Transfer Protocol), para enviar e receber arquivos.

Quando a contratação de uma hospedagem, o usuário recebe um usuário e senha para criar a conexão FTP. Esta conexão é feita através de programas como o Filezilla.

(Voltar)

O que é Linguagem de Programação

É uma linguagem escrita e formal que especifica um conjunto de instruções e regras usadas para gerar programas (software).

(Voltar)

Client-side

É a linguagem que vai rodar no client, e apesar de existirem várias, tem uma que roda tanto do lado server quanto do lado client. O Java Script.

O navegador mantem um módulo (no caso do Chrome é o V8) que é open Source, e serve interpretar e executar o Java Script. O V8 roda em um “navegador”

no server-side chamado node js.

(Voltar)

Server-side

A linguagem server-side roda no servidor, onde será feita toda lógica e processamento. Alguns exemplos são a PHP, C#, Java Script, Ruby, Java, dentre outros.

Cada uma tem a sua própria sintaxe, mas possui diversas semelhanças.

(Voltar)

HTML não é uma Linguagem de Progrmação

Não é uma linguagem de programação, mas sim um arquivo de marcação. Ele serve para criar textos delimitados que o navegador consiga entender, e cada nome destes

delimitadores tem uma função e comportamento específico.

(Voltar)

Falando sobre Tags

Fora as tags de estrutura básica do HTML, existem milhares de outras, cada uma com a sua função como por exemplo:

> i: Deixa o texto em itálico

>u: Deixa o texto sublinhado

>strong: Deixa o texto em negrito

>mark: Deixa o texto grifado

Existem tags que não tem a tag de fechamento, como por exemplo:

>input: Cria um campo digitáveis

>img: Adiciona imagem

(Voltar)

Atributos básicos das Tags

São propriedades que uma tag pode ter. Existem atributos genéricos e também alguns específicos de cada tag.

Genéricos:

>tag id = “exemplo”: Identifica o elemento

>tag stlyle = “exemplo”: Colocar comando CSS direto no elemento, como por exemplo “color: blue”

>tag class = “exemplo”: Também voltada o CSS, é uma forma de padronizar tudo o que tem haver com texto sinalizada, fazendo alterações iguais em tudo o que diz respeito a ele.

Específicos:

Input

>input type=“text”: Cria um campo digitável de letras

>input type=“number”: Cria um campo digitável de números

>input type=“color”: Cria um campo para escolha de cores

Img

>img src = “endereço de uma imagem”: Adiciona uma imagem

>img width = “tamanho desejado”: Dimensiona o tamanho da imagem

(Voltar)

Links

>a: Ancora em algum lugar

>a href = “página desejada”: Leva á página ou link desejado
